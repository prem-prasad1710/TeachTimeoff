# TeachTimeOff (Frontend)

This is a Vite + React frontend skeleton for the TeachTimeOff leave-management UI.

Features included in this scaffold:
- Dashboard page with stat cards and a chart (using Recharts)
- Profile page with user details and leave balances
- Header, Sidebar and Footer layout
- React Router wiring

Quick start

1. Install dependencies

```powershell
npm install
```

2. Start dev server

```powershell
npm run dev
```

Open http://localhost:5173
